Minio学习

发表于2023-07-21|更新于2023-07-29
|阅读量:

安装

https://dl.minio.io/server/minio/release/linux-amd64/minio

联网下载：

wget https://dl.minio.io/server/minio/release/linux-amd64/minio

安装wget

yum -y install wget

改权：

chmod 777 -R minio

后台运行：

nohup /usr/local/minio/minio server /home/file > /home/file/minio.log 2>&1 &

修改MINIO_ACCESS_KEY和MINIO_SECRET_KEY
export MINIO_ACCESS_KEY=fileadmin

export MINIO_SECRET_KEY=fileadmin

指定端口启动./minio server /home –console-address :9002 –address :9001
